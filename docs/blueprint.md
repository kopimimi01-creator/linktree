# **App Name**: Kopimi Experience

## Core Features:

- Hero Section Display: Display a full-width hero section with key headlines and call-to-action buttons to drive initial engagement.
- About Section Display: Display the values of Kopimi Cafe, handcrafted process, and welcoming ambiance, accompanied by relevant icons.
- Signature Experience Showcase: Showcase the unique aspects of Kopimi Cafe using cards and geometric icons.
- Menu Display: Display the menu categories (Coffee Series, Non-Coffee Series, Signature Blends, Seasonal Items) with a clean, text-only layout and subtle separators.
- Brand Story Presentation: Present the origins and mission of Kopimi Cafe, accompanied by abstract shape graphics.
- Call-to-Action Integration: Implement a call-to-action component with a rounded box, bold headline, and an 'Order Now' button to prompt user action.
- Footer Integration: Implement a footer with the Kopimi Cafe logo, social media links, operating hours, contact details, and copyright information.

## Style Guidelines:

- Primary color: Forest green (#0B3D2E) to set a warm, earthy tone.
- Background color: Soft beige (#E9DCC5), a lightly desaturated near-beige, for contrast and warmth.
- Accent color: White (#FFFFFF) for clean and modern elements.
- Headings: 'Literata' serif font for an elegant, slightly vintage look. Note: currently only Google Fonts are supported.
- Body: 'Inter' sans-serif font for a modern, clean feel. Note: currently only Google Fonts are supported.
- Use minimalist line icons for all sections. Avoid coffee-related visuals.
- Maintain a clean, structured layout with rounded components and generous spacing and padding.
- Implement smooth fade transitions and slight hover lift animations on buttons for an interactive user experience.
